RustyPanel Package Repository

Pre-compiled packages for RustyPanel.

Setup

# Import GPG key
curl -fsSL https://mirror-rustypanel.monity.io/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/rustypanel-archive-keyring.gpg

# Add repository (Ubuntu 24.04)
echo "deb [signed-by=/usr/share/keyrings/rustypanel-archive-keyring.gpg] https://mirror-rustypanel.monity.io noble main" | sudo tee /etc/apt/sources.list.d/rustypanel.list

# Update and install
sudo apt update
sudo apt install rustypanel-php85
